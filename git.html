راهنمای ساخت سایت در گیت هاب

تهیه توسط سید محمد پویا رضوی

این راهنما به شما مراحل ساخت یک سایت در گیت هاب را به صورت گام به گام توضیح می‌دهد:

مرحله 1: ایجاد مخزن (Repository) جدید

  1. به وب‌سایت GitHub بروید و وارد حساب کاربری خود شوید.
  2. روی دکمه New در گوشه بالا سمت راست صفحه کلیک کنید تا یک مخزن جدید ایجاد کنید.
  3. نامی برای مخزن خود انتخاب کنید. نام مخزن باید به صورت your-username.github.io باشد. (به جای your-username، نام کاربری گیت هاب خود را وارد کنید)
  4. مخزن را به صورت عمومی (Public) تنظیم کنید و گزینه Initialize this repository with a README را فعال کنید.
  5. روی دکمه Create repository کلیک کنید تا مخزن ایجاد شود.

مرحله 2: افزودن فایل‌های HTML، CSS و JavaScript

  1. به مخزن جدید خود بروید.
  2. روی دکمه Upload files کلیک کنید و فایل‌های HTML، CSS و JavaScript خود را آپلود کنید. فایل اصلی HTML باید نام index.html داشته باشد.
  3. پس از آپلود فایل‌ها، روی دکمه Commit changes کلیک کنید.

مرحله 3: فعال‌سازی GitHub Pages

  1. به بخش Settings مخزن بروید.
  2. به قسمت Pages در منوی سمت چپ بروید.
  3. در بخش Source، منبع خود را به main تنظیم کنید و دکمه Save را کلیک کنید.
  4. پس از چند لحظه، پیامی با لینک سایت خود مشاهده خواهید کرد. لینک شما باید به صورت https://your-username.github.io/ باشد.

تبریک!

شما با موفقیت یک سایت در گیت هاب ساختید. اکنون می‌توانید به راحتی سایت خود را با دیگران به اشتراک بگذارید.
